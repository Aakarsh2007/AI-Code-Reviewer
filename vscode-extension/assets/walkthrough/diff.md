# View AI Refactored Diff

After reviewing a file:

- Press **Ctrl+Shift+D** (Mac: **Cmd+Shift+D**) to open the diff viewer
- The left side shows your original code
- The right side shows the AI-optimized refactored version

You can also:
- **Copy** the refactored code: `Ctrl+Shift+C`
- **Apply** it directly to the file from the sidebar panel

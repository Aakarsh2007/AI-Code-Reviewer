# Review Your First File

Open any supported file (JS, TS, Python, Java, C++, Go, Rust, C#) and:

- Press **Ctrl+Shift+R** (Mac: **Cmd+Shift+R**)
- Or click the ⚡ button in the editor title bar
- Or right-click → **CodeSense AI: Review Current File**

Issues appear as inline diagnostics in the editor gutter and Problems panel.
